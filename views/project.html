Wall Mural

Updated February 22, 2014Posted by Kathryn Cassidy, Graphic Communication

 

Category: Art/Design

I really want to create a mural on Cal Poly's campus. I want it to be huge and in a place where everyone will walk by it a lot. I want it to be colorful, happy, and inspiring. I have experience with painting, but I want other artists to join in too!

Project Goals

When this project is completed, there will be a mural on Cal Poly's campus, as well as a timelapse video of the whole process.

Deadline/Timeline

This project is ongoing until it is done.
